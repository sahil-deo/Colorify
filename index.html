Colorify

About Usage Examples References

About

This API converts normal colors into simulated versions as perceived by users with Protanopia, Deuteranopia, and Tritanopia.
View Source Code

Built with Go, it’s designed to help developers test their websites and apps for color accessibility.

Developed by: Sahil Deo
GitHub | LinkedIn

Usage

Endpoint: GET /color?c=<hex1>&c=<hex2>...

Parameters:

  • c — One or more hex colors (e.g., #FF0000)

Response is JSON containing simulated color mappings.

Examples

Request:

GET http://localhost:8080/color?c=#FF0000&c=#00FF00

Response:

{
  "colors": {
    "protanopia": {
      "#FF0000": "#900000",
      "#00FF00": "#7FBF7F"
    },
    "deuteranopia": {
      "#FF0000": "#A00000",
      "#00FF00": "#7F997F"
    },
    "tritanopia": {
      "#FF0000": "#F20000",
      "#00FF00": "#6F9F9F"
    }
  }
}
